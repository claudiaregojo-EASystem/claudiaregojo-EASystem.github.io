The Foundation

One code.
Every kingdom.

The EA is a theoretical framework — a working hypothesis, not an established doctrine. Similar to how the Periodic Table organises elements by atomic structure, the EA organises homeopathic remedies by their position in the evolutionary hierarchy.

Every remedy represents an adaptive strategy shaped by evolutionary constraints. When we find the organism whose strategy mirrors the patient's own, we have found the simillimum.

The EA extends homeopathy's existing frameworks — Scholten, Sankaran, Yakir — into a single unified grammar. It is grounded in evolutionary biology and the Joshi boundary arc, giving practitioners a map that works across all kingdoms and all schools.

Clinical Application

How to Read a Case

Five structured steps move a patient's story into a precise evolutionary code.

01
Listen
Take the case without interruption. Record verbatim language, gestures, and recurring dreams.
02
Tag
Assign each expression to a code level — Kingdom through Species — using the reference tables.
03
Code
Build the 7-digit provisional code. Note alternative readings at each level.
04
Shortlist
Identify remedy families that match the full code profile. Consult materia medica to differentiate.
05
Validate
Repertorise rubrics to confirm. Let the code and repertory converge on the simillimum.
